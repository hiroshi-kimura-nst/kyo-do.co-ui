      CDokoittel

    行き先掲示板サービス　"ドコイッテル？"のUIです。

    環境構築

    初期セットアップ

    $ git clone https://github.com/tetsuya-zama/dokoit.tel-ui.git
$ cd dokoit.tel-ui
$ npm install

    ビルド

    $ webpack

    開発サーバの起動

    $ npm start
# http://localhost:8080でデバッグできる
# ソースを修正すると再ビルドして即反映してくれる
# 止めるときはctrl+c

    ドキュメント生成(esdoc)

    $ npm run esdoc
# ./esdocにドキュメントが生成される
