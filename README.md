# 行き先掲示板サービス　"ドコイッテル？"のUIです。
## 環境構築
### 初期セットアップ
```bash
$ git clone https://github.com/tetsuya-zama/dokoit.tel-ui.git
$ cd dokoit.tel-ui
$ npm install
```
### ビルド
```bash
$ webpack
```
###開発サーバの起動
```bash
$ npm start
# http://localhost:8080でデバッグできる
# ソースを修正すると再ビルドして即反映してくれる
# 止めるときはctrl+c (macの場合はcommand+c)
```
